# This document describes how to install cmake, fix-protocol log files extensions to colorer library (http://colorer.sourceforge.net/).

1. Put all files at <colorer dir>\hrc (for ex. c:\Program Files\Far2\Plugins\colorer-truemod\hrc)
2. Add "<location link="hrc/re_ext.hrc"/> " right after "<hrc-sets>" tag at <colorer dir>\catalog.xml
